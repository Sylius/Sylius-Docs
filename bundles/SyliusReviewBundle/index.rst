SyliusReviewBundle
==================

*SyliusReviewBundle* is a flexible and easy-to-manager reviews system for Symfony2 applications.

With this bundle you can configure what and how can be reviewed in your application. Each review consist of rating, title and comment, which allows to express opinion fully. Reviews can be confirmed or rejected
(for example, by administrator), that allows you to control your application's review system completely.
This bundle requires some configuration, but following documentation it's easy and fun.
